UC08 — Xem danh sách bài viết (View Posts)

Actor: User (Guest)

Mục tiêu: Xem list các bài public.

Precondition: Có posts with status=published.

Postcondition: Người dùng thấy danh sách, có link tới chi tiết.

Luồng chính:

Browser mở index.html.

Frontend fetch GET /api/posts?status=published&limit=...

API trả posts JSON.

Frontend render danh sách (thumbnail, title, summary).

Luồng phụ: Nếu API lỗi → show fallback message hoặc dùng data/posts.json.

Acceptance: List hiển thị, pagination/filters hoạt động.

Endpoint: GET /api/posts
