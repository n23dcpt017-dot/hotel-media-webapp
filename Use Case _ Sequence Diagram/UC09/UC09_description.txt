UC09 — Xem chi tiết bài viết (View Post Details)

Actor: User

Mục tiêu: Xem nội dung chi tiết, media, bình luận.

Precondition: Post published và có slug.

Postcondition: Nội dung hiển thị và comment hiển thị approved comments.

Luồng chính:

User click link → browser tới post.html?slug=....

Frontend gọi GET /api/posts?slug=xxx hoặc GET /api/posts/:id.

API trả post JSON + associated media + comments (approved).

Frontend render content, nhúng iframe video nếu có.

Luồng phụ: Nếu post not found → show 404 page.

Acceptance: Chi tiết bài hiển thị chính xác; video chạy; comments shown.

Endpoint: GET /api/posts?slug=, GET /api/comments?post_id=&status=approved
