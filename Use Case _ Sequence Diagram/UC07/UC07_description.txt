UC07 — Bắt đầu / Nhúng Livestream (Start Livestream)

Actor: Admin

Mục tiêu: Đưa livestream/clip lên site.

Precondition: Admin login; có stream URL / stream key (nếu WebRTC).

Postcondition: Livestream xuất hiện ở trang chủ/post.

Luồng chính:

Admin vào phần Livestream → nhập stream_type (youtube/tiktok/webrtc) và source (URL/ID).

Frontend gửi POST /api/media (type, source, title).

API lưu media record → trả success.

Frontend hiển thị iframe/ player dựa trên source.

Luồng phụ: URL không hợp lệ → validation error.

Acceptance: Livestream play được trên trang public.

Endpoint: POST /api/media, GET /api/media

DB fields: media {id, type, source, title, publish_at}
