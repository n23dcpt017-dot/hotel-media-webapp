UC12 — Xem livestream / clip ngắn (View Media)

Actor: User

Mục tiêu: Xem media list & play.

Precondition: Media records exist (type youtube/tiktok/webrtc).

Postcondition: Video phát trong iframe/player.

Luồng chính:

User vào media.html hoặc post detail.

Frontend GET /api/media hoặc post.media.

API trả list media.

Frontend render iframes.

Acceptance: Video/stream play; responsive.

Endpoint: GET /api/media
