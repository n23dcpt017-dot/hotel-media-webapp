UC03 — Chỉnh sửa bài viết (Edit Post)

Actor: Admin

Mục tiêu: Cập nhật bài có sẵn.

Precondition: Admin login, post tồn tại.

Postcondition: Post update lưu DB.

Luồng chính:

Admin chọn post từ list → click “Sửa”.

Frontend GET GET /api/posts/:id để lấy dữ liệu.

Hiển thị form có prefill → admin sửa → submit.

Frontend gửi PUT /api/posts/:id với payload.

API cập nhật DB → trả updated post.

Luồng phụ:

Nếu ID không tồn tại → trả 404.

Acceptance: Nội dung thay đổi phản ánh trên site.

Endpoint: GET /api/posts/:id, PUT /api/posts/:id

UI: admin.html (Edit modal/page)
